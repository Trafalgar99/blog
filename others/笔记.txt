
 VSCode编辑器？

    VS code下载地址：https://code.visualstudio.com/

    如何安装插件？  语言包、 open in browser  view in borwser

    学习编辑器基本使用？

        设置：文件-> 首选项 -> 设置  ( 大小、是否换行 word wrap )

        创建文件、创建文件夹、重命名和删除、搜索

        ctrl + s : 保存
        ctrl + a : 全选
        ctrl + x、ctrl + c、ctrl + v : 剪切、复制、粘贴
        ctrl + z、ctrl + y : 撤销、前进
        shift + end : 从头选中一行
        shift + home : 从尾部选中一行
        shift + alt + ↓ : 快速复制一行
        alt + ↑或↓ : 快速移动一行

        tab : 向后缩进
        tab + shift : 向前缩进

        多光标 : alt + 鼠标左键
        ctrl + d : 选则相同元素的下一个

PS快捷键（笔记）

1、图层的常规操作
    新建
    删除 delete
    多选：shift或者ctrl
    重命名 双击图层名称修改
    复制 ctrl+j
    编组 ctrl+g
    图层顺序

2、
    外部图片变换需要解除锁定
    自由变换 ctrl+t， 提交变换 enter， 取消 esc
    自由变换状态下，右键可以切换其他变换形态
    缩放，旋转，斜切，扭曲，透视， 变形

3、
    对选区填充 前景色 alt+delete  后景色ctrl+delete
    选取布尔运算 新建选取 选取相加 选取相减 选取相交
    取消选区 ctrl+d
    在选取模式下 右键可以设置羽化，能对边缘进行柔和处理
    反选快捷键 ctrl+shift+i

4、常用抠图工具
    索套工具 多边形索套工具 磁性索套工具 快速选择工具 魔棒工具

5、
    快速选择工具是常用抠图工具
    shift添加选取，alt是从选区里减去
    反选快捷键 ctrl+shift+i
    通过菜单栏 选择>修改>收缩 可以修正边缘

6、
    新建图层 （绘制的形状会在新的图层）
    合并形状  shift
    减去顶层形状    alf
    与区域形状相交
    排除重叠形状
    合并形状组件 （合并后无法分开）

7、
    钢笔形状工具可以绘制一些不规则形状（别忘了切换形状模式）
    白色箭头调整是锚点细节 黑色箭头是选择整个形状，可以进行移动
    常规形状工具可以绘制矩形，圆角矩形，圆，多边形，星形，自定义形状


8、
    污点修复画笔工具可以快速去除图片里面的污点
    修复画笔必须要按住alt键获取修复源，可以把获取的源置入到画笔的区域里，并与周围自动融合

9、
    内容识别必须绘制一段选曲，在右键填充里面找该工具
    仿制图章必须按住alt键获取仿制源

10、
    模糊能让局部进行模糊，锐化能让局部对比强烈，给人更清晰的视觉效果
    涂抹会让图片产生形态的改变，并带一定模糊效果

11、
    减淡加深主要对图片局部的调整
    海绵主要是去色与加色效果

12、
    蒙版里面白色代表可见，黑色代表不可见，灰色代表透明
    蒙版只有黑白灰三种颜色
    蒙版常常结合画笔对图片进行处理

13、
    浮雕效果是比较常用的图层样式，可以做出立体的效果
    描边可以给物体添加添加描边，描边可以设置成内描边，外描边，居中描边
    内阴影指的是元素内部的阴影效果
    内发光指的是元素内部的发光效果
